# clone the reop using:

git clone https://github.com/YanirMidler24/Country-Information-Portal

cd to the main dir

$ cd ./Country-Information-Portal


# Server Installation

cd ./server

## Install dependencies

$ npm install

## Running the server

Copy .env.example file to .env, fill all values that you need after

run command:
$ npm run start

watch mode

$ npm run start:dev




# Client Installation

cd ./client

## Install dependencies

$ npm install

or if you get errors

$ npm install --force

## Running the client

Copy .env.example file to .env, fill all values that you need after

run command:
$ npm run dev
