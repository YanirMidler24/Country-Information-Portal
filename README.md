# Server Installation

## Install dependencies

$ npm install

## Running the app

Copy .env.example file to .env, fill all values that you need after

run command:
$ npm run start

watch mode

$ npm run start:dev




# Client Installation

## Install dependencies

$ npm install

or if you get errors

$ npm install --force

Copy .env.example file to .env, fill all values that you need after

run command:
$ npm run dev
